--------Raiffeisen bank GUI ( CodingExercise Purpose only ) --------------

	Project done with Java Swing 
	Null Layout with setBounds coordinates (x, y, w, h )
	User data ( Client information ) - dataBase package
	A simple GUI for Swing and code organisation exercise
	Frames: 
	-main Login-frame
		-Check the data validation from the userData class ( dataBase package )
	
	-Options-frame
		-Only deposit option works 
		-Same functionality for all users 
	
	-Deposit-frame 
		-Ammount storing 
		-Feedback in the console after ammount registration

	-Upcoming features: 

	* More option functionality in the Option-frame
	* Different ammounts accessed when logged in from the dataBase package 
	* Exceptions & pop-up errors
	* minor details and etc.
